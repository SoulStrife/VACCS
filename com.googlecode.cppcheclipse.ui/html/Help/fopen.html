fopen()

The C library function fopen() creates a stream using a filename The prototype is

FILE * fopen(const char *filename, const char * mode)

filename
Name of the file to be opened
mode
File modes as defined here.

One of the potential problems with fopen() is that it could be susceptible to symbolic link attacks. For example, 

int main(int argc, char* argv[]) {
  //The file /tmp/temp.txt could be a symbolic link from /tmp/temp.txt to /etc/passwd (or some other system critical file)
  FILE *f = fopen("/tmp/temp.txt", "w");

  //If this program was ran as root, it would overwrite /etc/passwd with this
  fputs("Not trying to overwrite anything", f);

  fclose(f);

  return 1;
}
