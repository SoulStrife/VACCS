fdopen()

The C library function fdopen() creates a stream using an existing file descriptor. The prototype is

int fdopen(int fd, const char * mode)

fd
File descriptor
mode
File modes as defined here. The mode of the stream must be compatible with the mode of the file descriptor. The file position indicator of the new stream will be reset to the beginning of the file belonging to fd. Error and end-of-file indicators are also reset. No automatic truncation of the file will be performed.

The potential problems with fdopen() are very similar with the problems with fopen()
