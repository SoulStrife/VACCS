freopen()

The C library function freopen() associates a new filename with the given stream and closes the old file. The prototype is

FILE *freopen(const char *filename, const char * mode, FILE *stream)

filename
Name of the file to be opened
mode
File modes as defined here.
stream
Reference to a FILE object that is to have its open file replaced

The potential problems with freopen() are very similar with the problems with fopen()
