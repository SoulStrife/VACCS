fread()

The C library function fread() attempts to read nmemb*size bytes from the given stream into the array pointed to by ptr The prototype is

size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream)

ptr
Pointer to the block of memory that will be read into
size
The size in bytes of each element to be read
nmemb
Number of elements, each with size of size bytes
stream
The input stream.
