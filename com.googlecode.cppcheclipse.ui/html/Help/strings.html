Strings are a fundamental concept in C, but they are not a built-in type like integers and characters. C strings area actually an array of characters terminated by first null character ('\0'). As an example, the code

char *s = "SECURITY"

declares the variable s as a pointer to the memory location that begins the string "SECURITY" as shown in the following figure

The value of a string is the sequence of the values of the contained characters, in order, without the null character. Thus, the value of the above string is SECURITY. The length of a string is the number of bytes preceding the null character. In the above example, the length of the string is 8.
