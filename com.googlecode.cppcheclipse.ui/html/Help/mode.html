modes

"r"
Opens a file for reading. File must already exist
"w"
Creates an empty file for writing. Truncates existing file if it exists.
"a"
Opens a file so that it can be appended to. All writing operations will only append to the end of the file. File created if id doesn't exist
"r+"
Opens file for both reading and writing. The file must exist
"w+"
Creates a new file for reading and writing
"a+"
Opens a file for both reading and appending
