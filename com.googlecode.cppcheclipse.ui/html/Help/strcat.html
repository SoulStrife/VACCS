strcpy()

The C library function strcat() concatenates two strings (See here for a quick tutorial on C strings) from one memory location to another. The prototype for strcpy is

char *strcat(char *dest, char* src)

strcat() appends the string src to the string dest. The null termination character at the end of dest is overwritten with the first character of src and a null character is added to the end of the concantenated string.

Because strcpy() does not check to make sure that the concatenating the two strings does not overrun the memory allocate for dest. Consider the following program.

static char a[4];
static char b[4]]

int main(int argc, char **argv) {

  strcpy(a,"one");
  strcpy(b,"two");
  strcat(a,b);

  printf("a = %s\n",a);
  printf("b = %s\n",b);

}
